## Overview
An 8-bit style web game where you run a pizza shop and shoot pizza ingredients to fulfill customer orders. Earn money and upgrade your weapon and stats in the shop!

## Gameplay
- Talk to customers and take their order
- Shoot pizza topppings to collect necessary ingredients (pepperoni, cheese, mushrooms, olives, peppers, ham, pineapple, and more)
- Fulfill customer orders
- Earn upgrades and survive as long as possible

## Controls
-WASD to move
-Enter to talk to customers
-Space to shoot

## Project Structure
- `Overworld.js` / `OverworldMap.js` — core game world and map logic
- `Person.js` / `GameObject.js` — player and game object handling
- `Ingredient.js` + individual topping files — ingredient logic
- `OrderManager.js` — customer order system
- `HUD.js` / `UpgradeMenu.js` — heads-up display and upgrades
- `Bullet.js` — shooting mechanics
- `sprites/` / `backgrounds/` — game assets
